## 모듈과 제어
- 많이들 작성한 거 미리 가져오겠다 import or from import 로 가져오기
- 내장 모듈
  설치없이 불러오기 가능  
  math, random, datatime
- 'as' 키워드로 2개 이상의 모듈에서 동일한 거 불러올 때 구분가능
  ```python
  from math import pi
  from my_math import pi as my_pi
  ```
- 될 수 있으면 import로 가져오기(스타일가이드)
