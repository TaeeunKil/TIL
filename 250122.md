#### map
- 순회 가능한 데이터 구조(iterable)의 모든 요소에 함수를 적용하고 결과를 map으로 반환
  ```
  numbers = [1, 2, 3]
  result = map(str, numbers)
  print(result)  # <map object at 0x00000239C915D760>
  print(list(result))  # ['1', '2', '3']
  ```
