- 추가적인 반응형 CSS
  - breakpoint 외의 작업은?(300px 이 때 조작하고 싶다)
  - 색깔을 바꾸고 싶다
  - 미디어 쿼리(@media) 사용법 : @media 미디어타입 and (조건)
      ```  
      @media screen and (min-width:500px) {
        .header{
        background-color : red;
        }
      }
      ```
    - min-width 조건 이상일 때
    - max-width 조건 이하일 때
    - orientation 화면의 가로 세로 여부(portrait(세로), landscape(가로))
  - transition : 변할 때 부드럽게 전환
